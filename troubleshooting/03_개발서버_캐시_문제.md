# 개발 서버 캐시 문제

## 문제 상황
코드를 수정하고 저장했는데 **이유 없는 에러가 발생**하거나, 브라우저에 변경사항이 반영되지 않는 현상이 발생했습니다.

## 원인
Webpack Dev Server의 캐시가 꼬여서 이전 빌드 결과가 계속 사용되는 현상으로 예상

## 해결 방법

### 1. 개발 서버 재시작 (가장 빠름)
```bash
# 터미널에서 Ctrl + C로 서버 종료
# 다시 실행
npm start
```

### 2. 캐시 완전 삭제 (근본 해결)
```bash
# 서버 종료 후
rm -rf node_modules/.cache
npm start
```

### 3. 브라우저 캐시도 함께 삭제
```
브라우저에서:
Ctrl + Shift + R (강력 새로고침)
또는
F12 → Network 탭 → "Disable cache" 체크
```


## 핵심
**변경사항이 반영 안되면 → 서버 재시작!**
